How this fits together

This Admin page and the Cases page write to your browser's local storage — they're meant to be opened locally (or on a private/internal link) by the L&D team, not the public GitHub Pages link. After uploading data here, click Export updated dataset and commit the downloaded JSON files into assets/data/ in the repo, then push — that's what refreshes the public Overview and Cohort pages. See the README for the recommended internal-hosting options for Admin & Cases.
